# Cypress Sandbox
A set of tools for test automation.

https://github.com/MDeakin/cypress_sandbox

# Requirements:
- npm

# Installation instructions:
```
npm install
```
- OR -
```
npm i -g cypress
```
```
npm install --save-dev cypress
```
# Run:
```
$(npm bin)/cypress open
```
# Update:
```
npm update -g cypress
```
